Today, I am creating my first blog post on my website. Along with this, is a program soon coming to this website for download.  
This Program is called _**Steam Beta Switcher**_. It is for the Popular Digital Distribution Platform called **Steam**. My Program allows you to keep your old installation of any game when you switch to a beta branch, and then switch back and forth without any hassle.  
It works by scraping search results on the Steam Online Store Page, to obtain its "_appid_", which is a very important Identifier we will need later on. Then, it uses this "_appid_" to get info on the games beta branches. It then displays the name of each beta branch to the user, and allows them to choose which one to switch to/install. When a selection is made, the program runs a command through CLI utilities provided by Steam. This allows the game to updated. The program also renames the game's installation directory to prevent it from being overwritten.
